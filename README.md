# Health Resource Intelligence System (HRIS)
---

## AI-powered patient segmentation for smart healthcare resource allocation

* Problem *
Hospitals rely on manual reporting
No predictive view of patient demand
Resource shortages (drugs, staff, equipment)
Sudden disease spikes are hard to manage
# 👉 Result: inefficient treatment + wasted resources

* Solution *
We built an AI system that:
Groups patients using machine learning
Detects health demand patterns
Converts clusters into resource recommendations

* How it works *
Upload patient Excel/CSV
AI cleans and processes data
Patients are clustered using KMeans
System interprets clusters
Generates hospital action plan

* Output Example *
Malaria-heavy cluster → increase anti-malarial stock
Pediatric respiratory cluster → increase oxygen supply
High-visit chronic patients → allocate more staff

* Impact *
Better resource allocation
Faster outbreak detection
Reduced drug shortages
Data-driven hospital planning

# Tech Stack
Flask (backend)
Scikit-learn (ML)
Pandas (data processing)
Chart.js (visualization)
PWA (offline usability)
